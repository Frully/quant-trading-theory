# 如何确定仓位？(How to Size Positions?)

> 核心问题：给定一个有正期望的信号，应该下注多少？

---

## 问题的本质

仓位管理是将 Alpha 转化为收益的关键环节。错误的仓位可以让正期望的策略亏损。

**不解决会怎样？**
- 下注过大：破产风险
- 下注过小：收益不足以覆盖成本
- 仓位波动过大：心理压力导致决策失误

# 量化交易理论研究 (Quantitative Trading Theory)

## 项目简介

本项目致力于量化金融基础理论的系统性研究与整理。聚焦于理论本身（What/Why），而非操作方法（How）。

## 如何阅读

```
00. 入门（什么是量化交易）
     ↓
01. 市场有效性（EMH：基础框架）
     ↓
02-04. Alpha来源（为什么存在超额收益）
     ↓
05-11. 策略理论（策略为什么有效）
     ↓
12-14. 定价理论（收益如何定价）
     ↓
15-17. 组合/风险/执行（如何实现收益）
     ↓
18-19. 市场生态（Alpha的演化）
```

每篇文章结构：
- **理论研究（80%）**：是什么、核心内容、实证证据、学术争议
- **实践启示（20%）**：对量化交易的指导意义

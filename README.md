# 量化交易理论研究 (Quantitative Trading Theory)

## 项目简介

本项目致力于量化金融基础理论的系统性研究与整理。涵盖市场效率理论、资产定价、行为金融、风险管理、执行算法等核心领域。

## 目录结构

```
quant-trading-theory/
├── core/                   # 核心理论文档
│   └── foundational_theories.md  # 量化交易理论框架
├── literature/             # 文献阅读笔记
├── notes/                  # 研究笔记（灵感、推导、概念辨析）
├── drafts/                 # 写作草稿（完整文章、论文）
├── templates/              # 标准化模板
└── references.md           # 核心参考文献
```

## 核心内容

**[理论框架](core/foundational_theories.md)** 涵盖：

1. **市场效率之争** — EMH vs AMH
2. **方法论基础** — 统计学习、因果推断
3. **定价与配置理论** — 一价定律、MPT、APT、衍生品定价
4. **Alpha 来源** — 行为金融、微观结构、限制套利
5. **风险管理** — VaR/CVaR、动态风控
6. **跨学科视角** — 信息论、复杂系统
7. **执行与工程** — 最优执行、交易成本、策略生命周期

## 如何使用

1. 从 `core/foundational_theories.md` 了解整体理论框架
2. 在 `literature/` 中添加文献阅读笔记
3. 使用 `templates/` 中的模板记录研究过程
4. 将成熟的想法整理到 `drafts/` 形成完整文章

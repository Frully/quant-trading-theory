```
00. 入门（什么是量化交易）
     ↓
01. 市场有效性（EMH：基础框架）
     ↓
02-04. Alpha来源（为什么存在超额收益）
     ↓
05-07. 定价理论（收益如何定价）
     ↓
08-10. 组合/风险/执行（如何实现收益）
     ↓
11-12. 市场生态（Alpha的演化）
```

每篇文章结构：
- **理论研究（80%）**：是什么、核心内容、实证证据、学术争议
- **实践启示（20%）**：对量化交易的指导意义

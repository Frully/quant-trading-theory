# 均值回归策略

> **类型**：交易策略（逆向策略）
> **核心逻辑**：价格偏离均值后会回归——跌多了买入，涨多了卖出
> **理论基础**：[价值与反转策略](../core/06_value_and_reversal.md)
> **与趋势跟踪的关系**：逻辑相反——价格回归 vs 趋势延续

---

## 核心概念定义

### 什么是均值回归？

**均值回归 (Mean Reversion)** 是一种交易策略，核心假设是：**价格偏离"正常水平"后，会倾向于回归到这个水平**。

这个概念最早由英国统计学家 Francis Galton 在 1886 年提出，他在研究人类身高时发现：**极端高或极端矮的父母，其子女的身高往往会"回归"到人群平均值**。这个现象后来被广泛应用于金融市场。

均值回归交易者相信：
- 价格会围绕某个"均值"上下波动
- 极端的偏离是暂时的，终将修正
- 跌得越多，反弹的概率越大；涨得越多，回调的概率越大

**举例**：

```
某股票的 20 日均价是 100 元
当前价格跌到 85 元（偏离 -15%）

均值回归交易者的逻辑：
    "价格偏离均值太远了"
    "应该会反弹回 100 元附近"
    → 买入

如果价格确实回到 100 元：
    盈利 = (100 - 85) / 85 = 17.6%
```

### 与趋势跟踪的对比

均值回归和趋势跟踪是两种**逻辑完全相反**的策略：

| 维度 | 均值回归 | 趋势跟踪 |
|-----|---------|---------|
| **核心假设** | 价格会回归均值 | 趋势会延续 |
| **买入时机** | 价格下跌后 | 价格上涨后 |
| **卖出时机** | 价格上涨后 | 价格下跌后 |
| **操作逻辑** | 低买高卖（逆向） | 追涨杀跌（顺势） |
| **最佳市场** | 震荡市/区间市 | 趋势市 |
| **最差市场** | 单边趋势市 | 震荡市 |
| **收益分布** | 高胜率、偶尔大亏（左偏） | 低胜率、偶尔大盈（右偏） |

**关键认知**：没有哪个策略绝对优越，它们适用于不同的市场状态。

### 与网格交易、马丁格尔的区别

这三种策略都基于"价格会回来"的假设，但有重要区别：

| 维度 | 均值回归 | 网格交易 | 马丁格尔 |
|-----|---------|---------|---------|
| **理论基础** | 统计学、行为金融 | 无理论基础 | 无理论基础 |
| **入场条件** | 统计检验（协整、ADF） | 任意价格间隔 | 任意价格 |
| **仓位管理** | 固定或反向调整 | 等量加仓 | 加倍加仓 |
| **止损机制** | 有（价差失效时止损） | 通常没有 | 没有 |
| **风险特征** | 可控 | 较高 | 极高（爆仓风险） |

**核心区别**：

- **均值回归**：首先通过统计检验确认"均值确实存在且稳定"，然后才交易
- **网格交易**：假设价格会回来，但不检验这个假设是否成立
- **马丁格尔**：不仅假设价格会回来，还在亏损时加倍下注

均值回归策略的关键是**先验证再交易**，而不是盲目假设。

## 一、理论基础

### 1.1 为什么价格会均值回归？

这是均值回归策略的核心问题。如果不理解"为什么"，就无法判断"什么时候"均值回归会失效。

**学术理论基础**：

De Bondt & Thaler (1985) 首次系统提出均值回归假设，核心观点是"涨多了会跌，跌多了会涨"。Fama & French (1988) 研究股票收益的记忆模式，发现过去的价格确实可以预测未来价格。

**行为金融解释**：

```
投资者过度反应
    ↓
好消息 → 过度乐观 → 价格涨过头
坏消息 → 过度悲观 → 价格跌过头
    ↓
情绪修正
    ↓
价格回归合理水平
```

**具体机制**：

1. **投资者情绪波动**：市场情绪导致股票被过度追捧或厌恶，创造了回归的条件

2. **前景理论 (Prospect Theory)**：Kahneman & Tversky (1979) 发现，投资者对损失的敏感度高于收益，导致系统性的定价偏差

3. **流动性需求**：大型机构的买卖会暂时推动价格偏离均值，但这种偏离不可持续

**与随机游走的争论**：

均值回归的反对者认为价格遵循**随机游走**——未来价格变动不可预测。

| 支持均值回归 | 支持随机游走 |
|------------|------------|
| Fama & French (1988) | Fama (1970) 有效市场假说 |
| Poterba & Summers (1988) | Lo & MacKinlay (1988) |
| De Bondt & Thaler (1985) | Malkiel《漫步华尔街》 |

**市场现实**：一句流传甚广的警告——

> "市场保持非理性的时间，可能比你保持偿付能力的时间更长。"
>
> —— 凯恩斯

这意味着：即使价格最终会回归，你也可能在回归之前破产。

### 1.2 Ornstein-Uhlenbeck 过程

**Ornstein-Uhlenbeck (OU) 过程**是描述均值回归的核心数学模型：

$$dX_t = \theta(\mu - X_t)dt + \sigma dW_t$$

**变量含义**：

| 符号 | 含义 | 说明 |
|-----|------|------|
| $X_t$ | 当前价格 | 我们观察到的价格 |
| $\mu$ | 长期均值 | 价格回归的目标 |
| $\theta$ | 回归速度 | 越大回归越快 |
| $\sigma$ | 波动率 | 随机波动的幅度 |
| $dW_t$ | 维纳过程 | 随机扰动项 |

**直观理解**：

```
当价格 X > 均值 μ 时：
    μ - X < 0
    → dX 倾向于负值
    → 价格倾向于下跌（向均值回归）

当价格 X < 均值 μ 时：
    μ - X > 0
    → dX 倾向于正值
    → 价格倾向于上涨（向均值回归）
```

**与几何布朗运动的区别**：

股价通常用几何布朗运动 (GBM) 建模，GBM **没有**回归中枢——价格可以无限上涨或下跌。而 OU 过程有明确的回归目标。

### 1.3 半衰期

**半衰期 (Half-Life)** 是衡量均值回归速度的关键指标：

$$\text{Half-life} = \frac{\ln(2)}{\theta} \approx \frac{0.693}{\theta}$$

**含义**：价格偏离均值后，回归一半所需的预期时间。

**举例**：

```
假设某配对的半衰期 = 10 天
当前价差偏离均值 = 20 元

预期：
    10 天后，价差约为 10 元（回归了一半）
    20 天后，价差约为 5 元
    30 天后，价差约为 2.5 元
```

**实践意义**：

| 半衰期 | 策略含义 |
|-------|---------|
| < 5 天 | 高频策略，适合日内或隔日交易 |
| 5-20 天 | 短期策略，持仓数天到数周 |
| 20-60 天 | 中期策略，持仓数周 |
| > 60 天 | 长期策略，或不适合均值回归交易 |

**警告**：半衰期的估计有很大不确定性。即使有超过 10,000 个数据点，估计仍可能不准确。

### 1.4 检验方法

在交易之前，必须先检验均值回归是否存在。

**ADF 检验 (Augmented Dickey-Fuller Test)**：

检验时间序列是否**平稳**（stationary）。平稳序列具有均值回归特性。

```
原假设 H0：序列有单位根（非平稳，即随机游走）
备择假设 H1：序列平稳（存在均值回归）

检验步骤：
1. 对序列进行回归：ΔX_t = α + βX_{t-1} + ... + ε_t
2. 检验 β 是否显著小于 0
3. 若 p 值 < 0.05，拒绝 H0，认为存在均值回归
```

**Hurst 指数**：

另一种衡量序列记忆性的方法：

| Hurst 值 | 含义 |
|---------|------|
| H = 0.5 | 随机游走，无法预测 |
| H < 0.5 | 均值回归，反转特性 |
| H > 0.5 | 趋势延续，动量特性 |

**协整检验 (Cointegration Test)**：

用于配对交易。两个序列各自可能是随机游走，但它们的线性组合可能是平稳的。

```
例子：
    可口可乐股价：随机游走
    百事可乐股价：随机游走

    可口可乐 - β × 百事可乐 = 平稳序列（协整）

    → 可以交易这个"价差"
```

**Engle-Granger 两步法**（1987）：
1. 对两个序列回归，获取残差
2. 对残差进行 ADF 检验

**Johansen 检验**：更稳健，可以同时处理多个序列。

## 二、学术证据与收益特征

### 2.1 短期反转效应

**Jegadeesh (1990) & Lehmann (1990)** 发现了短期反转效应：

- **样本**：1934-1987 年美国股票
- **发现**：过去 1 周到 1 个月的输家，在接下来 1 个月跑赢赢家
- **收益**：月均超额收益约 **2%**

**解释理论**：

1. **过度反应假说**：投资者对短期信息过度反应，随后修正

2. **流动性供给假说**（Stefan Nagel, 2012）：
   - 短期反转收益是**流动性供给的补偿**
   - 做市商在别人恐慌卖出时买入，承担风险
   - 反转收益与 VIX 高度相关——高波动时反转更强

**重要警告**：短期反转效应在学术上存在，但**实际交易很难获利**——交易成本可能吃掉全部收益（详见第四节）。

### 2.2 配对交易研究

**Gatev, Goetzmann & Rouwenhorst (2006)**：配对交易的经典研究

- **样本**：1962-2002 年美国股票
- **方法**：选择历史价格走势最相似的股票对
- **结果**：年化超额收益可达 **11-12%**

**关键发现——收益率在下降**：

| 时期 | 月均超额收益 |
|-----|------------|
| 1989 年之前 | 118 bp (1.18%) |
| 1989 年之后 | 38 bp (0.38%) |
| 下降幅度 | **约 2/3** |

**Do & Faff (2010)** 进一步确认：
- 配对交易收益持续下降
- 但在**危机期间**（2001-2002、2007-2009）收益显著**回升**
- 这说明均值回归策略具有一定的"危机保护"特性

**为什么收益在下降？**

```
1990s-2000s：
    量化交易崛起
    更多资金追逐相同的机会
    价差更快被消除
    → 收益被套利掉
```

### 2.3 统计套利：Avellaneda & Lee

**Avellaneda & Lee (2010)** 在 *Quantitative Finance* 发表了统计套利的重要研究：

**方法**：
- 使用 **PCA（主成分分析）** 提取市场因子
- 交易个股相对于因子的**残差**
- 残差被建模为 OU 过程

**结果**：

| 时期 | PCA 策略夏普比率 | ETF 策略夏普比率 |
|-----|-----------------|-----------------|
| 1997-2002 | **1.44** | 1.1 |
| 2003-2007 | 0.9 | 0.7 |

**关键发现**：
- 收益在 **2003 年后显著下降**
- 加入**交易量信息**后，2003-2007 年夏普比率提升至 **1.51**
- 这表明策略需要不断演进

### 2.4 ETF 均值回归

**IBS 效应 (Internal Bar Strength)**：

研究发现，在股票 ETF 中：
- 添加 IBS 过滤器可提升 **10 个百分点**的收益
- 同时减少 **45%** 的市场暴露时间

**SPY/QQQ RSI 策略**（QuantifiedStrategies 研究）：

| 策略 | 年化收益 | 胜率 | 夏普比率 |
|-----|---------|------|---------|
| 买入持有 | 6.39% | - | - |
| RSI(2) 策略 | 10.01% | - | - |
| RSI(2) < 10 入场 | **12.7%** | **75%** | **2.85** |

**特点**：在熊市（2000-2002、2008/09、2018、2020 年 3 月）表现优异。

## 三、策略类型

### 3.1 技术指标类

**布林带 (Bollinger Bands)**：

```
中轨 = N 日移动平均线
上轨 = 中轨 + K × 标准差
下轨 = 中轨 - K × 标准差

常用参数：N = 20, K = 2

信号：
    价格触及下轨 → 做多（超卖）
    价格触及上轨 → 做空（超买）
    价格回归中轨 → 出场
```

**举例**：

```
某股票：
    20 日均价 = 100 元
    20 日标准差 = 5 元

    上轨 = 100 + 2×5 = 110 元
    下轨 = 100 - 2×5 = 90 元

当价格跌到 90 元时：
    触发买入信号
    目标：回到 100 元
    预期收益：11%
```

**RSI (Relative Strength Index)**：

```
RSI = 100 - 100/(1 + RS)
RS = 平均上涨幅度 / 平均下跌幅度

常用参数：14 日 RSI

传统信号：
    RSI < 30 → 超卖，买入
    RSI > 70 → 超买，卖出
```

**重要警告**：

> RSI 均值回归策略在**强趋势市场中会失效**。
>
> 在牛市中，RSI 可能长期处于 70 以上；在熊市中，可能长期处于 30 以下。
>
> 盲目按 RSI 逆向交易会被趋势碾压。

**解决方案**：加入趋势过滤器（如 200 日均线）。

### 3.2 配对交易

配对交易是均值回归最经典的应用。

**距离法**（Gatev 等人使用）：

```
步骤：
1. 选择股票对：历史价格走势最相似的两只股票
2. 标准化价格：将两只股票价格标准化（均值 0，标准差 1）
3. 计算距离：两个标准化价格序列的平方和
4. 选择距离最小的 N 对

交易规则：
    价差 > 2σ → 做空价差（买入低价股，卖空高价股）
    价差 < -2σ → 做多价差（买入高价股，卖空低价股）
    价差回归 0 → 平仓
```

**协整法**：

```
步骤：
1. 检验协整：用 Johansen 或 Engle-Granger 方法
2. 估计对冲比率：Y = β × X + 残差
3. 构建价差：Spread = Y - β × X
4. 计算 Z-score：Z = (Spread - 均值) / 标准差

交易规则：
    Z > 2 → 做空价差
    Z < -2 → 做多价差
    Z 穿越 0 → 平仓
```

**Z-score 阈值设定**：

| 阈值类型 | 入场 | 出场 | 特点 |
|---------|------|------|------|
| 保守 | |Z| > 2.5 或 3 | |Z| < 0.5 | 信号少，单笔收益高 |
| 激进 | |Z| > 1.5 或 2 | |Z| < 0.5 | 信号多，胜率高 |
| 常用组合 | |Z| > 2 | |Z| < 1 | 平衡 |

**举例**：

```
可口可乐 (KO) 和百事可乐 (PEP)

协整回归：KO = 0.8 × PEP + 残差
价差 = KO - 0.8 × PEP

当前：
    KO = 60 元
    PEP = 70 元
    价差 = 60 - 0.8×70 = 4 元
    历史价差均值 = 0 元
    历史价差标准差 = 2 元
    Z-score = (4 - 0) / 2 = 2

信号：做空价差
    卖出 1 股 KO（$60）
    买入 0.8 股 PEP（$56）
    净投入 = $4

如果价差回归到 0：
    KO = 56 元，PEP = 70 元
    价差 = 56 - 56 = 0
    平仓盈利 = 60 - 56 = $4（KO 部分）
    收益率 = $4 / $4 = 100%
```

### 3.3 统计套利

**PCA 方法**（Avellaneda & Lee）：

```
思路：
1. 用 PCA 提取市场的主要因子（如市场、行业、规模）
2. 计算每只股票的"预期收益"（基于因子暴露）
3. 实际收益 vs 预期收益的残差 = 特异性偏离
4. 交易残差的均值回归

步骤：
1. 对股票收益矩阵做 PCA
2. 提取前 K 个主成分作为因子
3. 回归：R_i = β_1×F_1 + β_2×F_2 + ... + ε_i
4. 残差 ε_i 应该均值回归
5. ε_i 偏高 → 做空；ε_i 偏低 → 做多
```

**ETF/成分股套利**：

```
思路：ETF 价格应该等于其成分股的加权平均价格

当 ETF 价格 > 成分股价值：
    卖出 ETF
    买入成分股
    → 赚取差价

当 ETF 价格 < 成分股价值：
    买入 ETF
    卖出成分股
    → 赚取差价
```

### 3.4 跨资产策略

**期货日历价差**：

```
做多远月合约，做空近月合约
利用价差的均值回归特性

例子（原油）：
    近月合约：80 美元
    远月合约：82 美元
    价差：2 美元

    历史价差均值：1 美元
    当前价差偏高 → 做空价差
        卖出远月，买入近月

    如果价差回归到 1 美元：
        盈利 = 2 - 1 = 1 美元/桶
```

**研究发现**：WTI 原油和天然气的日历价差策略夏普比率可超过 **2**。

**波动率均值回归 (VIX)**：

波动率具有天然的均值回归特性——不会永远很高，也不会永远很低。

```
VIX 期货曲线：
    正常情况（升水）：远月 > 近月
    恐慌情况（贴水）：远月 < 近月

策略：
    VIX 很高且期货贴水 → 做空波动率（卖 VIX 期货）
    VIX 很低且期货升水 → 谨慎或等待
```

**风险**：VIX 飙升时做空损失巨大（如 2018 年 2 月"波动率末日"）。

## 四、交易成本的关键影响

### 4.1 为什么交易成本如此重要？

均值回归策略通常需要**频繁交易**，这使得交易成本成为决定盈亏的关键因素。

学术研究中看起来可观的收益，在实际交易中可能被成本吃掉。

### 4.2 成本构成

| 成本类型 | 说明 | 典型水平 |
|---------|------|---------|
| **买卖价差** | 买价与卖价的差 | 股票 0.01-0.1%，加密货币 0.1-0.5% |
| **滑点** | 实际成交价与预期价的差 | 取决于订单大小和流动性 |
| **市场冲击** | 大单对价格的影响 | 大单可能显著移动价格 |
| **佣金手续费** | 经纪商/交易所费用 | 0.01-0.1% |

**举例**：

```
配对交易：买入 A，卖出 B
    买入 A：支付买卖价差 0.05%
    卖出 B：支付买卖价差 0.05%
    佣金：0.02%

    单边成本 = 0.05% + 0.05% + 0.02% = 0.12%
    往返成本 = 0.12% × 2 = 0.24%

如果预期收益是 0.5%：
    净收益 = 0.5% - 0.24% = 0.26%
    成本吃掉了近一半的收益！

如果交易 100 次：
    毛收益 = 100 × 0.5% = 50%
    总成本 = 100 × 0.24% = 24%
    净收益 = 26%
```

### 4.3 学术研究发现

**Kaul & Nimalendran (1990)**：
- NASDAQ 股票的短期反转**主要源于买卖价差误差**
- 不是真正的价格反转，而是买卖价差的"弹跳"

**Avramov, Chordia & Goyal (2006)**：
- 短期反转与**流动性高度相关**
- 逆向策略的利润**小于交易成本**

**Leung & Li (2015)** 研究最优交易时机：
- **交易成本越高，最优入场价格越极端**
- 必须等到更大的偏离才值得交易
- 止损水平越高，投资者会在更低的止盈水平主动平仓

**结论**：独立的短期反转策略因换手率过高而**不切实际**。

### 4.4 解决方案

**1. 聚焦大市值、高流动性品种**：

de Groot, Huij & Zhou (2012) 证明：
- 只交易大股票可显著降低成本
- 经过改进后，反转策略扣除成本后仍可获得每周 **30-50bp** 的收益

**2. 降低换手率**：

- 使用更宽的交易阈值（如 Z > 2.5 而非 Z > 2）
- 将反转筛选纳入其他策略的自然再平衡过程
- 不要频繁进出

**3. 使用限价单**：

```
市价单：立即成交，但付出买卖价差
限价单：可能不成交，但价格更优

对于非紧急的均值回归交易，限价单通常更好
```

**4. 选择流动性好的品种**：

- 买卖价差窄
- 日均交易量高
- 市场深度足够

## 五、加密货币应用

加密货币市场为均值回归提供了独特的机会和挑战。

### 5.1 比特币的特殊性

**学术发现：比特币不存在简单的均值回归**

2013-2019 年的数据显示，比特币表现为"均值发散"而非均值回归——价格趋势性强。

但存在**非对称均值回归**（Corbet & Katsiampa, 2020）：

| 收益方向 | 回归特性 |
|---------|---------|
| 负收益 | 回归更强、更快 |
| 正收益 | 持续性更高，回归较慢 |

**含义**：下跌后反弹的概率高于上涨后回调的概率。

**组合策略**：

```
趋势跟踪在局部最高点效果好：
    当 BTC 创 20 日新高 → 继续做多（趋势延续）

均值回归在局部最低点效果好：
    当 BTC 创 10 日新低 → 买入（反弹预期）

组合：趋势 + 回归
```

### 5.2 RSI 在比特币上的表现

**关键发现：传统 RSI 均值回归在比特币上效果差**

回测结果：
- RSI(14) 和 RSI(2) 的均值回归策略表现都很差
- 2019 年两种策略都不盈利（因为价格趋势性强）
- RSI 可能长时间停留在超买/超卖区域

**什么有效：动量型 RSI**

| 策略 | 年化收益 | 最大回撤 |
|-----|---------|---------|
| 买入持有 | 101% | 83% |
| RSI(5) 动量策略 | **122%** | **39%** |

RSI 动量策略规则：
- 入场：5 日 RSI 突破 50 向上
- 出场：5 日 RSI 跌破 50

**教训**：在比特币上，RSI 更适合作为动量指标而非均值回归指标。

### 5.3 加密货币配对交易

**研究发现**（Binance 上 26 种加密货币）：

| 频率 | 月均收益 |
|-----|---------|
| 日频 | -0.07% |
| 1 小时 | +2.3% |
| 5 分钟 | **+11.61%** |

**结论**：高频策略在加密货币配对交易中表现显著更好。

**常用交易对**：

- **BTC/ETH**：最常被研究的协整对
- 可构建包含 BTC、ETH、BCH、LTC 的协整组合

**交易示例**：

```
BTC-ETH 价差偏离均值 1.5σ：

    BTC 被低估 → 做多 BTC
    ETH 被高估 → 做空 ETH

    价差收敛（Z-score 穿越 0）时平仓

    示例收益：13.6%
```

### 5.4 资金费率套利

这是加密货币特有的均值回归机会。

**机制**：

永续合约没有到期日，通过**资金费率**机制与现货价格锚定：

```
永续价格 > 现货价格 → 资金费率为正 → 多头付给空头
永续价格 < 现货价格 → 资金费率为负 → 空头付给多头
```

**套利策略**：

```
当资金费率为正时：
1. 买入现货
2. 做空等量永续合约
3. 头寸对冲，市场中性
4. 收取资金费率（通常每 8 小时结算）
```

**收益与风险**：

| 指标 | 数值 |
|-----|------|
| 6 个月最高收益 | 115.9% |
| 最大可能损失 | 1.92% |
| 年化收益（高资金费率期） | > 10% |

**均值回归特性**：
- 基差通常遵循均值回归过程
- 正常市场中基差回归的半衰期平均为 **4-6 小时**
- 资金费率呈现"粘性"，高费率往往会持续

**风险管理**：
- 基差风险：现货和永续价格可能极端分化
- 流动性风险：波动时期难以平仓
- 建议杠杆不超过 **2-3 倍**，维持 **150%** 保证金缓冲

### 5.5 跨交易所与 DeFi 套利

**跨交易所套利**：

不同交易所的价格可能存在差异：

```
Binance BTC 价格：30,000 USDT
Coinbase BTC 价格：30,050 USDT

套利：
    在 Binance 买入 BTC
    在 Coinbase 卖出 BTC
    利润：50 USDT/BTC

限制：
    转账时间
    提币手续费
    资金占用
```

**研究发现**：
- 套利机会在网络拥堵和价格波动时出现
- 交易量和链上活动增加会提高交易所间相关性，减少套利机会

**DeFi 中的均值回归**：

去中心化交易所 (DEX) 使用自动做市商 (AMM)，核心公式：**x × y = k**

```
套利机制：
    DEX 价格偏离外部市场
        ↓
    套利者在 DEX 和 CEX 之间交易
        ↓
    DEX 价格回归均衡
```

**流动性提供者的风险**：
- **无常损失 (Impermanent Loss)**：代币价格变化导致的损失
- 如果价格回到初始位置，损失会消失
- 但如果在价格分化时提取流动性，损失就永久化了

## 六、风险与陷阱

### 6.1 LTCM 案例（1998）

**Long-Term Capital Management (LTCM)** 是均值回归策略失败的教科书案例。

**背景**：
- 由两位诺贝尔奖得主（Scholes 和 Merton）创立
- 核心策略：债券套利，基于收敛交易
- 杠杆率：约 **30 倍**

**策略逻辑**：

```
买入"低估"债券（如新兴市场债券）
卖空"高估"债券（如美国国债）

假设：
    价差会均值回归
    两种债券的价值最终会趋同

公司名称的由来：
    "Long-Term"（长期）= 相信价差长期会收敛
```

**早期成功**：

| 年份 | 年化收益（扣费后） |
|-----|-----------------|
| 1994 | 21% |
| 1995 | 43% |
| 1996 | 41% |

**灾难（1998 年 8 月）**：

```
俄罗斯债务违约
    ↓
全球投资者恐慌
    ↓
涌入美国国债（安全资产）
    ↓
LTCM 的价差不但没有收敛，反而急剧扩大
    ↓
所有头寸同时亏损
    ↓
8 月单月亏损 44%
    ↓
流动性枯竭，无法平仓
    ↓
濒临破产
```

**结果**：美联储协调 14 家银行注资 36 亿美元救助 LTCM，避免系统性风险。

**核心教训**：

> **"耐心资本"只有在你最不需要它有耐心的时候才有耐心。**

1. **均值回归策略不能高杠杆**：30 倍杠杆 + 价差扩大 = 灾难
2. **市场危机时流动性会消失**：你想卖的时候卖不掉
3. **极端事件会打破历史关系**：俄罗斯违约是"不可能"事件
4. **时间风险**：价差可能在回归前把你拖垮

### 6.2 黑天鹅与体制转变

**黑天鹅风险**：

```
均值回归假设：价格偏离均值后会回归

黑天鹅打破这个假设：
    极端事件 → 均值本身发生变化
    → 你一直在等待回归到一个不存在的均值
```

**体制转变 (Regime Shift)**：

| 变化类型 | 例子 |
|---------|------|
| 行业颠覆 | 智能手机淘汰诺基亚 |
| 政策变化 | 利率从正转负 |
| 技术革命 | AI 重估科技股估值 |
| 公司丑闻 | 财务造假被揭露 |

**例子**：

```
2015 年之前：
    A 公司和 B 公司高度协整
    它们的价差稳定在 0 附近

2015 年：
    A 公司的主营业务被新技术颠覆
    A 公司股价持续下跌
    价差不断扩大

均值回归交易者：
    "价差太大了，应该回归"
    不断买入 A，卖出 B
    不断亏损
    最终爆仓

正确做法：
    监控协整关系是否仍然成立
    定期重新检验
    设置止损
```

### 6.3 常见失败模式

**1. 均值漂移**：

```
问题：你以为的"均值"已经变了

例子：
    过去 5 年，某股票的 P/E 均值是 15
    你在 P/E = 10 时买入，等待回归到 15

    但实际上：
        行业进入衰退期
        合理 P/E 已经下降到 8
        P/E 继续从 10 跌到 6

    你亏损 40%
```

**2. 杠杆与仓位过大**：

均值回归策略的收益分布特点：
- **高胜率**：70-80% 的交易盈利
- **偶尔大亏**：少数交易亏损很大

```
危险的逻辑：
    "胜率这么高，加大仓位"
    "这次肯定会回归，加仓"

结果：
    一次大亏抹掉所有利润
    高杠杆 + 大亏 = 爆仓
```

**3. "Hold and Hope"（持有并祈祷）**：

```
最危险的行为：
    亏损后不止损
    "再等等，会回来的"
    继续加仓

结果：
    小亏变大亏
    大亏变爆仓

正确做法：
    设置止损
    承认这次判断错误
    保留资金等待下次机会
```

## 七、与动量策略的关系

### 7.1 时间尺度分布

均值回归和动量在**不同时间尺度**上主导：

| 时间尺度 | 主导效应 | 解释 |
|---------|---------|------|
| 极短期（1 周-1 月） | **反转** | 流动性冲击、过度反应 |
| 中期（3-12 月） | **动量** | 信息缓慢扩散、行为偏差 |
| 长期（3-5 年） | **反转** | 过度反应修正、估值回归 |

**实践含义**：

```
如果你做短期交易（日-周）：
    均值回归可能更有效

如果你做中期交易（月-季）：
    动量可能更有效

如果你做长期投资（年）：
    价值/反转可能更有效
```

### 7.2 收益分布互补

两种策略的收益分布正好**互补**：

| 维度 | 均值回归 | 动量 |
|-----|---------|------|
| **胜率** | 高（70-80%） | 低（35-45%） |
| **盈亏比** | 低（<1） | 高（>2） |
| **收益偏度** | 左偏（偶尔大亏） | 右偏（偶尔大盈） |
| **最佳市场** | 震荡市 | 趋势市 |

**组合优势**：

```
均值回归：
    □□□□□□□□□□ 小盈利
    □□□□□□□□□ 小盈利
    □□□□□□□□ 小盈利
    ■■■■■■■■■■■■■■■ 大亏损

动量：
    ■■ 小亏损
    ■■ 小亏损
    ■■ 小亏损
    □□□□□□□□□□□□□□□ 大盈利

组合：
    互相对冲
    降低整体波动
    提高风险调整收益
```

**Menkhoff 等人的研究**（外汇市场）：

| 策略 | 年化收益 |
|-----|---------|
| 单独动量策略 | 11.14% |
| 结合反转信号的双排序策略 | **20.24%** |

### 7.3 市场状态切换

**隐马尔可夫模型 (HMM)** 可以识别市场状态：

```
状态识别：
    状态 1：趋势市场（动量有效）
    状态 2：震荡市场（均值回归有效）

策略切换：
    如果当前是状态 1 → 使用动量策略
    如果当前是状态 2 → 使用均值回归策略
```

**研究发现**：
- HMM 在识别市场状态变化方面表现最佳
- 但存在**滞后问题**，可能错过拐点
- 状态切换本身有成本

**Hurst 指数应用**：

```
计算滚动 Hurst 指数：
    H < 0.45 → 使用均值回归
    H > 0.55 → 使用趋势跟踪
    0.45 < H < 0.55 → 混合或观望
```

## 核心结论

1. **均值回归的本质是"逆向思维"**：在别人恐慌时买入，在别人贪婪时卖出

2. **必须先验证再交易**：不是所有东西都会均值回归——用 ADF、协整检验确认

3. **交易成本是生死线**：学术收益不等于实际收益，高频交易成本可能吃掉全部利润

4. **时间是最大的敌人**：价格可能在回归前把你拖垮——设置止损，控制杠杆

5. **与动量策略互补**：不同时间尺度、不同市场状态，两者可以组合

6. **加密货币有独特机会**：资金费率套利、高频配对交易，但风险也更高

7. **LTCM 是永恒的警示**：高杠杆 + 均值回归 = 灾难配方
